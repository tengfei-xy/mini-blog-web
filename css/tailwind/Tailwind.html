尺寸
文档：https://tailwind.nodejs.cn/docs/background-size
使用 bg-auto 以默认大小显示背景图片。
使用 bg-cover 缩放背景图片，直到它填满背景层。
使用 bg-contain 将背景图片缩放到外边缘，而无需裁剪或拉伸。
