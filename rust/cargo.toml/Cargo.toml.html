[dependencies]
structopt = "0.3.26"

[target]
[[bin]]
name="ff"
path = "src/main.rs"

从代码中访问Cargo.toml的值
访问方式
fn main() {
    let package_version = env!("CARGO_PKG_VERSION");
    println!("Package version: {}", package_version);

    let package_authors = env!("CARGO_PKG_AUTHORS");
    println!("Package authors: {}", package_authors);
}
具体变量
  • CARGO_PKG_NAME:包名
  • CARGO_PKG_VERSION: 包的版本号。
  • CARGO_PKG_AUTHORS: 包的作者列表，用逗号分隔。
  • CARGO_PKG_DESCRIPTION: 包的描述信息。
  • CARGO_PKG_HOMEPAGE: 包的主页 URL。
  • CARGO_PKG_REPOSITORY: 包的代码仓库 URL。
  • CARGO_PKG_LICENSE: 包的许可证。
  • CARGO_PKG_LICENSE_FILE: 包的许可证文件路径。
  • CARGO_PKG_KEYWORDS: 包的关键词列表，用逗号分隔。
  • CARGO_PKG_TARGET_*: 这些环境变量用于访问特定目标的信息，其中 * 代表目标名称。例如，CARGO_PKG_TARGET_X86_64_UNKNOWN_LINUX_GNU 可以用于访问 Linux 上 x86_64 架构的目标的相关信息。
