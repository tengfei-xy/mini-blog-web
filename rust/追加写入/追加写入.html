use std::fs::OpenOptions;
use std::io::Write;

fn main() {
    let mut file = OpenOptions::new()
        .write(true)
        .append(true)
        .open("myfile.txt")
        .unwrap();

    let variables = vec!["one", "two", "three"];

    for variable in variables {
        file.write_all(format!("{} ", variable).as_bytes()).unwrap();
    }

    file.write_all(b"\n").unwrap();
}
在这个例子中，我们使用 OpenOptions 结构体来配置文件打开选项，包括写入和追加。然后，我们使用 open 方法打开文件，并将其分配给一个可变的 File 对象。
接下来，我们定义了一个包含多个字符串的向量 variables。
在循环中，我们使用 write_all 方法将每个字符串写入文件。write_all 方法接受一个字节切片作为参数，因此我们需要使用 format! 宏来将字符串转换为字节切片。
最后，我们使用 write_all 方法写入一个换行符，以分隔每一行内容。
我们使用 unwrap 方法来处理任何可能发生的错误。如果文件打开或写入操作失败，unwrap 将导致程序崩溃并打印错误信息。
这样，我们就将多个变量的值追加写入到文件中，并使用换行符分隔每一行内容。
