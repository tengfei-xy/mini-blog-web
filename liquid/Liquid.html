文档：官方文档、
定义：是 Shopify 的模板语言，是一种简单的标记语言
作用：在 Shopify 主题中创建动态内容
用途：开发者可以创建可根据特定条件或数据动态更改的主题
场景
  • 在产品页面中显示产品名称、描述和价格。
  • 根据客户的位置显示不同的内容。
  • 根据产品库存创建“缺货”或“有货”消息。
  • 创建可根据客户偏好进行定制的个性化电子邮件。
语法
  • 变量：用双大括号括起来，例如 {{ product.title }}。
  • 条件语句：使用 {% if %}, {% else %}, {% endif %} 语法。
  • 循环：使用 {% for %}, {% endfor %} 语法。
  • 过滤器：用于格式化或转换变量值，例如 {{ product.price | money }}。
示例
根据产品库存显示“缺货”或“有货”消息：
{% if product.available %}
  <span class="in-stock">有货</span>
{% else %}
  <span class="out-of-stock">缺货</span>
{% endif %}
创建一个根据客户位置显示不同内容的下拉菜单
<select>
  {% for country in countries %}
    <option value="{{ country.code }}">{{ country.name }}</option>
  {% endfor %}
</select>
