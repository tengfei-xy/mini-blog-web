windows、office激活

第一步：在Linux上部署kms服务器

clone项目
git clone git@github.com:Wind4/vlmcsd.git
指定项目位置
mv vlmcsd-master /usr/local/vlmcsd
进入vlmcsd项目路径
cd /usr/local/vlmcsd
安装依赖
yum install gcc git make -y
编译项目
[root@node21 vlmcsd]# make
