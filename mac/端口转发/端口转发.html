socat
安装socat
brew install socat
将访问本机的57204，转发到192.168.0.197:57205
sudo socat TCP4-LISTEN:57204,fork TCP4:192.168.0.197:57205
通过 SOCKS 代理
socat TCP4-LISTEN:<本地端口>,reuseaddr,fork SOCKS:<代理服务器IP>:<远程地址>:<远程端口>,socksport=<代理服务器端口>
通过 HTTP 代理
socat TCP4-LISTEN:<本地端口>,reuseaddr,fork PROXY:<代理服务器IP>:<远程地址>:<远程端口>,proxyport=<代理服务器端口>
gost
本地监听
启动http代理服务器
gost -L=http://0.0.0.0:21211
作为socks5代理服务器
gost -L=socks5://0.0.0.0:21211
启动socks5h代理服务器
gost -L=socks5h://0.0.0.0:21211
本地监听并转发
Socks5，对外部地址访问本机的8080，将转发到127.0.0.1:57204
gost -L=:8080 -F=socks5://127.0.0.1:57204
tcp转发
gost -L tcp://:21212 -F tcp://192.168.0.52:21211
http转发
gost -L=:21211 -F=socks5://192.168.0.52:21211
