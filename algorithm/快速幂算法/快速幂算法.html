快速幂算法（Fast Exponentiation）是一种用于高效计算指数幂的算法。它通过利用指数的二进制表示形式来减少乘法和幂运算的次数，从而提高计算速度。
传统的幂运算算法需要进行多次乘法操作，例如计算 x^n，需要执行 n-1 次乘法。但是快速幂算法可以将指数 n 表示为二进制形式，例如 n = b[0] * 2^0 + b[1] * 2^1 + b[2] * 2^2 + ... + b[k] * 2^k，其中 b[i] 表示二进制表示的 n 的第 i 位。
利用二进制表示，我们可以通过迭代的方式快速计算幂，而无需进行多次乘法。具体步骤如下：
  1. 初始化结果 result 为 1，将指数 n 转换为二进制形式。
  2. 从二进制表示的最低位开始，依次遍历每一位：
    • 如果当前位为 1，将 result 与底数 x 相乘。
    • 将底数 x 自乘，即 x = x * x。
  3. 将当前位向高位移动一位。
  4. 重复步骤 2 和 3，直到遍历完整个二进制表示。
  5. 最终的 result 即为 x 的指数幂 x^n。
通过这种方法，快速幂算法的时间复杂度可以降低到 O(log n)，相比传统的幂运算算法具有更高的效率。这使得快速幂算法在需要频繁进行指数幂计算的场景下非常有用，例如密码学、图论和数论等领域。
