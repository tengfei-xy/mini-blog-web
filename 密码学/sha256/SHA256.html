定义：单向散列函数的改进版本
黑盒原理：将任意长度的数据转换为一个256位的二进制字符串。该字符串称为哈希值或摘要
作用：验证数据的完整性和真实性
性质：抗碰撞性、抗第二原像性
长度：160位
安全性：未破解
sha1和sha256的区别
  • 哈希值长度: SHA1的哈希值长度为160位，而SHA256的哈希值长度为256位。这意味着SHA256的哈希值空间更大，因此更不容易发生碰撞。
  • 安全性: SHA1已经被证明存在安全漏洞，可以被攻击者利用来伪造数据。而SHA256目前还没有发现任何安全漏洞，因此被认为是更安全的哈希算法。
  • 速度: SHA256的计算速度比SHA1更快。
  • 并行性: SHA256可以更容易地并行化，从而提高计算速度。

    fn sha256(&self) -> String {
        // create a Sha256 object
        let mut hasher = Sha256::new();

文件
use sha2::Sha256;
use std::fs::File;
use std::io::Read;

fn main() {
    let mut file = File::open("myfile.txt").unwrap();
    let mut hasher = Sha256::new();

    // 将文件内容更新到哈希状态中
    let mut buf = [0; 4096];
    loop {
        let n = file.read(&mut buf).unwrap();
        if n == 0 {
            break;
        }
        hasher.update(&buf[..n]);
    }

    // 生成最终的哈希值
    let hash = hasher.finalize();

    // 打印哈希值
    println!("SHA256 hash of myfile.txt: {:?}", hash);
}
