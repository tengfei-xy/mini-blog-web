定义：是一种前端构建工具
作用：用于构建 Vue.js 项目，也可以用于构建其他 JavaScript 项目
特点：快速、简单、灵活
创建项目
一般创建
npm create vite@latest
搭建一个 Vite + Vue 项目
npm create vite@latest my-vue-app -- --template vue
通过社区模板创建项目,通过degit工具假设项目位于 GitHub 上并用作 main 默认分支
npx degit user/project#main my-project
安装包
npm install
运行在测试
Vite 将会监视您的文件更改，并自动更新浏览器中的内容。
npm run dev
发布到生产环境
npm run build
配置
添加代理
export default defineConfig({
  plugins: [vue()],
  server: {
    proxy: {
      '/api': {
        target: 'http://localhost:3000',
        changeOrigin: true,
      },
    },
  },
})
公共基础路径 / base
参数
默认参数：/
  • 绝对 URL 路径，例如 /foo/
  • 完整的 URL，例如 https://foo.com/
  • 空字符串或 ./（用于嵌入形式的开发）
参数位置
// vite.config.js
import { defineConfig } from 'vite'

export default defineConfig({
  base: '/foo/' // 开发或生产环境服务的公共基础路径
})
