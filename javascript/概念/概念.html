es模块
名称：ECMAScript
定义：ECMAScript 是 JavaScript 语言的标准化版本，是 ECMAScript 6 中引入的一种模块化系统
作用：用于将 JavaScript 代码组织成独立的模块。
特点
  • 模块是独立的文件，可以包含 JavaScript 代码、HTML 代码、CSS 代码等。
  • 模块之间可以相互依赖，通过 import 语句导入其他模块的成员。
  • 模块可以被浏览器原生支持，无需使用其他模块化工具。
使用示例
在 HTML 中，使用 es 模块需要在 script 标签中设置 type 属性为 "module"：
<script type="module" src="my-module.js"></script>
my-module.js 是一个 es 模块，可以使用 import 语句导入其他模块的成员：
import { add } from "math";

console.log(add(1, 2)); // 3
