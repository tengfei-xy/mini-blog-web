定义：箭头函数是 JavaScript 中于 ES6 中引入的的一种新函数语法
使用方法：箭头函数使用 => 运算符将参数与函数体连接起来。
特点
简洁的语法：箭头函数的语法比传统函数更简洁，尤其是在只有一个参数或返回值的情况下。
词法作用域：箭头函数中的 this 关键字指向其定义时的上下文，而不是调用时的上下文。
不绑定 arguments 对象：箭头函数没有 arguments 对象。
不能用作构造函数：箭头函数不能用作构造函数。
适用场景
  • 作为回调函数传递给其他函数
  • 在数组、对象等数据结构上进行操作
  • 定义简短的函数
最佳实践
  • 对于只有一个参数或返回值的简单函数，使用箭头函数。
  • 在需要使用 this 关键字的函数中，使用箭头函数。
  • 在需要避免创建 arguments 对象的函数中，使用箭头函数。
示例
// 只有一个参数和返回值的箭头函数
const double = n => n * 2;

// 具有多个参数的箭头函数
const add = (a, b) => a + b;

// 具有多行函数体的箭头函数
const filter = arr => arr.filter(n => n % 2 === 0);
