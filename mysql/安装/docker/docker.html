docker镜像地址
container-registry.oracle.com/mysql/community-server:8.0
启动容器
启动后查看root密码 sudo docker logs mysql8 2>&1 | grep GENERATED | awk '{print $5}'
一般启动
sudo docker run --name=mysql8 --restart on-failure -d container-registry.oracle.com/mysql/community-server:8.0
挂载外部文件
sudo docker run --name=mysql80new \
   --mount type=bind,src=/path-on-host-machine/my.cnf,dst=/etc/my.cnf \
   --mount type=bind,src=/path-on-host-machine/datadir,dst=/var/lib/mysql \
-d container-registry.oracle.com/mysql/community-server:8.0
首次登录
sudo docker exec -it mysql8 mysql -uroot -p
首次登录并修改root密码，立即生效
ALTER USER 'root'@'localhost' IDENTIFIED BY 'password';
