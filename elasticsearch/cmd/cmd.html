字段
所有字段
GET my_index/my_type/1
单个字段
GET my_index/my_type/1?stored_fields=name,age
搜索
GET /elasticsearch/_search
{
  "query":{
    "match": {
      "_id": "know-external"
    }
  },
  "_source": ["property"]
}
`
动态数据类型
设置字段的数据类型
PUT my-index/_mapping/_doc
{
  "_doc": {
    "_include_type_name": true,
    "properties": {
      "my-key": {
        "type": "text"
      }
    }
  }
}

PUT elasticsearch/_mapping
{
  "properties": {
    "check": {
      "type": "object"
    }
  }
}
启动
PUT my-index
{
  "mappings": {
    "my-type": {
      "dynamic": "true"
    }
  }
}
