语法
表（table）
说明
  • 表由方括号（[]）括起来。
  • 方括号内的内容表示表的名称。
  • 表名称可以是单个的标识符，也可以是用点号分隔的多个标识符，形成层级结构。
  • 表可以包含键值对，也可以包含其他表。
表示
# author是表
[author]
name = "John Doe"
email = "johndoe@example.com"
注释
说明
  • 使用#符号开头的行表示注释。
  • 注释可以出现在任何位置，可以是单独的一行，也可以在键值对的后面。
表示
# 这是一个注释
键值对
说明
  • 键值对由键、等号（或冒号），以及值组成。
  • 键和值之间使用等号（=）或冒号（:）分隔。
  • 键和值可以使用双引号或单引号括起来，也可以不使用引号。
  • 键值对可以出现在顶层，也可以嵌套在表（table）中。
表示
# 键值对为 timeout = 500
[database]
timeout = 5000
数组
说明
  • 数组由方括号（[]）括起来。
  • 数组元素可以是任意类型的值，包括字符串、整数、浮点数、布尔值等。
  • 数组元素之间使用逗号（,）分隔。
表示
一维
fruits = ["apple", "banana", "orange"]
多维
matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
]

toml库
直接获取
首先，将以下代码添加到你的 Cargo.toml 文件中，以添加 toml 依赖项：
[dependencies]
toml = "0.8.10"
接下来，创建一个新的 Rust 文件，例如 main.rs，并添加以下代码：
extern crate toml;

use std::fs::File;

fn main() {
    // 打开 TOML 文件
    let contents = fs::read_to_string("expample.toml").expect("读取失败");

    // 解析 TOML 字符串
    let parsed_toml = content.parse::<toml::Value>().expect("解析 TOML 失败");

    // 访问解析后的 TOML 数据
    let value = parsed_toml["some_key"].as_str().unwrap();

    println!("解析的 TOML 值: {}", value);
}
在上面的代码中，我们首先打开了一个名为 config.toml 的 TOML 文件。然后，我们将文件内容读取到一个字符串中，并使用 parse 方法将其解析为 toml::Value 类型的对象。最后，我们可以通过索引和类型转换等方式访问解析后的 TOML 数据。
解析单个字段
添加依赖
[dependencies]
toml = "0.8.10"
然后，在您的Rust代码中，您可以使用toml库来解析Toml文件。下面是一个简单的示例：
extern crate toml;

use std::fs::File;

fn main() {
    // 打开Toml文件
    let contents = fs::read_to_string("expample.toml").expect("读取失败");
    
    // 解析Toml文件
    let parsed_toml = contents.parse::<toml::Value>().expect("无法解析Toml文件");
    
    // 对解析结果进行操作
    // 例如，获取特定键的值
    let value = parsed_toml.get("key").expect("找不到键").as_str().expect("无效的值");
    println!("键的值为: {}", value);
    
    // 读取msyql节的user值
    // let user = parsed_toml["mysql"]["user"].as_str().unwrap();
}
上面的代码首先打开Toml文件，并将其内容读取到一个字符串中。然后，使用parse方法解析Toml字符串，将其转换为toml::Value类型的值。您可以根据需要对parsed_toml进行操作，例如获取特定键的值。
解析到结构体
添加依赖项
[dependencies]
serde = {version="1.0.197", features=["derive"]}
toml = "0.8.10"
定义toml文件
[proxy]
enable = true
address = "104.224.178.90:21211"
target = ["dync"]
创建一个新的 Rust 文件，并添加以下代码：
use serde::Deserialize;
use std::fs;

#[derive(Debug, Deserialize)]
struct ProxyConfig {
    enable: bool,
    address: String,
    target: Vec<String>,
}

#[derive(Debug, Deserialize)]
struct Config {
    proxy: ProxyConfig,
}

fn main() {
    // 打开 TOML 文件
     let contents = fs::read_to_string("config.toml").expect("无法读取文件");

    // 解析 TOML 字符串到结构体
    let parsed_toml: Config = toml::from_str(&content).expect("解析 TOML 失败");

    // 访问解析后的数据
    println!("解析的 TOML 值: {:?}", parsed_toml);
}
