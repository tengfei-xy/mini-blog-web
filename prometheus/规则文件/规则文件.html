语法
groups:
  [ - <rule_group> ]
<rule_group>
# 用于标识一个告警规则的名称。它可以用来在规则列表中查找特定的规则，也可以用来在 alertmanager 中定义规则的优先级。
name: <string>

# How often rules in the group are evaluated.
[ interval: <duration> | default = global.evaluation_interval ]

# Limit the number of alerts an alerting rule and series a recording
# rule can produce. 0 is no limit.
[ limit: <int> | default = 0 ]

rules:
  [ - <rule> ... ]
<rule>
expr：expr 是一个 PromQL表达式，用于计算指标数据的值。
记录规则
作用：record 规则用于将计算结果保存为新的指标数据。
输出：record 规则将计算结果保存为新的指标数据
语法
# The name of the time series to output to. Must be a valid metric name.
record: <string>

# The PromQL expression to evaluate. Every evaluation cycle this is
# evaluated at the current time, and the result recorded as a new set of
# time series with the metric name as given by 'record'.
# 定义：expr 是一个 PromQL 表达式，用于计算指标数据的值。
expr: <string>

# Labels to add or overwrite before storing the result.
labels:
  [ <labelname>: <labelvalue> ]
示例
将 CPU 使用率的平均值保存为新的指标数据：
groups:
- name: my_job
  rules:
  - record: cpu_usage_average
    expr: avg(cpu_usage_percent)
警报规则
作用：alert 规则用于触发告警。
输出：alert 规则将告警信息发送到报警系统
语法
# The name of the alert. Must be a valid label value.
alert: <string>

# The PromQL expression to evaluate. Every evaluation cycle this is
# evaluated at the current time, and all resultant time series become
# pending/firing alerts.
expr: <string>

# Alerts are considered firing once they have been returned for this long.
# Alerts which have not yet fired for long enough are considered pending.
[ for: <duration> | default = 0s ]

# How long an alert will continue firing after the condition that triggered it
# has cleared.
[ keep_firing_for: <duration> | default = 0s ]

# Labels to add or overwrite for each alert.
labels:
  [ <labelname>: <tmpl_string> ]

# Annotations to add to each alert.
annotations:
  [ <labelname>: <tmpl_string> ]
示例
当 CPU 使用率超过 80% 时触发告警：
groups:
- name: my_job
  rules:
  - alert: MyJobCPUUsageHigh
    expr: cpu_usage_percent > 80
    for: 1m
    labels:
      severity: critical
    annotations:
      description: "CPU 使用率超过 80%"
