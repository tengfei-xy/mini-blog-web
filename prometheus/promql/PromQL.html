操作符
  • 算术运算符：+、-、*、/、%
  • 比较运算符：=、!=、<、<=、>、>=
  • 逻辑运算符：&&、||、!
函数
聚合函数
sum、avg、min、max
数学函数
sin、cos、tan、log
字符串函数
length、substring、replace
示例
  • 计算 CPU 使用率的平均值：avg(cpu_usage_percent)
  • 计算 CPU 使用率的最大值：max(cpu_usage_percent)
  • 计算 CPU 使用率的变化率：delta(rate(cpu_usage[5m]))
  • 计算 CPU 使用率是否超过 80%：cpu_usage_percent > 80
  • 计算 CPU 使用率是否在 70% 到 80% 之间：cpu_usage_percent >= 70 && cpu_usage_percent <= 80
