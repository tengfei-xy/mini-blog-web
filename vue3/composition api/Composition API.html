作用：用于在组件中创建和管理响应式状态、计算属性和侦听器。
意义：提供了一种更灵活、更可重用的方式来编写 Vue 组件。
基本API
defineProps()
Vue版本要求高于等于3.2
作用：定义组件的属性。用于在 setup 函数中定义一个组件的 props。
返回：一个 reactive 对象，这个对象包含了传递给组件的 props。
示例
通过defineProps() 定义了一个名为 msg 的 prop，它的类型是 String。然后我们可以在 setup 函数中访问这个 prop。
<script setup>
import { defineProps } from 'vue'

const props = defineProps({
  msg: String
})

console.log(props.msg) // 打印传递给组件的 msg prop
</script>

  • defineEmits()：定义组件的事件。
  • inject()：注入依赖项。
  • provide()：提供依赖项。
响应式 API
作用：用于创建和管理响应式状态。
  • ref()：创建一个响应式引用。
  • reactive()：创建一个响应式对象。
  • toRef()：将响应式值转换为响应式引用。
  • toRefs()：将响应式对象转换为响应式引用的对象。
计算属性 API
作用：用于创建计算属性。
  • computed()：创建一个计算属性。
  • watch()：监视一个响应式值的变化。
生命周期 API
作用：用于在组件的生命周期中执行钩子函数
  • onMounted()：在组件挂载后执行。
  • onUpdated()：在组件更新后执行。
  • onBeforeUnmount()：在组件卸载前执行。
