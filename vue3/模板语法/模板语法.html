说明：Vue使用基于HTML的模板语法
意义：能够声明式地将其组件示例的数据绑定到呈现的DOM上
底层：Vue会将模板编译成高度优化的JS代理。结合响应式系统，当应用状态变更时，Vue能够智能推导出需要重新渲染的组件的最少数量，并应用最少的DOM操作
通过在 app.config.globalProperties 上显式地添加它们，供所有的 Vue 表达式使用。
指令
定义：指令是带有 v- 前缀的特殊 attribute
参数
定义：在指令名后通过一个冒号隔开做标识
静态参数
示例一
名称：href
作用：告诉 v-bind 指令将表达式 url 的值绑定到元素的 href attribute 上。
简写：参数前的一切 (例如 v-bind:) 都会被缩略为一个 : 字符。
<a v-bind:href="url"> ... </a>

<!-- 简写 -->
<a :href="url"> ... </a>
示例二
名称：click。
作用：是要监听的事件名称：
简写：v-on 有一个相应的缩写，即 @ 字符。
<a v-on:click="doSomething"> ... </a>

<!-- 简写 -->
<a @click="doSomething"> ... </a>
动态参数
说明：在参数上使用JS表达式，需要包括在一堆方括号内
值限制：应是字符串或null。null会移除绑定，非字符串会触发警告 语法限制：空格和引号是不合法的，比如<a :['foo' + bar]="value"> ... </a>
示例一
说明：这里的 attributeName 会作为一个 JavaScript 表达式被动态执行，计算得到的值会被用作最终的参数。举例来说，如果你的组件实例有一个数据属性 attributeName，其值为 "href"，那么这个绑定就等价于 v-bind:href。
<a v-bind:[attributeName]="url"> ... </a>

<!-- 简写 -->
<a :[attributeName]="url"> ... </a>
示例二
说明：将一个函数绑定到动态的事件名称上，当 eventName 的值是 "focus" 时，v-on:[eventName] 就等价于 v-on:focus。
<a v-on:[eventName]="doSomething"> ... </a>

<!-- 简写 -->
<a @[eventName]="doSomething"> ... </a>
修饰符
定义：以点开头的特殊猴嘴，表明指令需要以特殊方式被绑定
数据绑定形式
文本
使用方式：支持单一表达式的双大括号
说明：双大括号的标签会替换为相应组件示例msg属性的值，同时每次msg属性更改时他会同步更新。
示例
<span>Message: {{ msg }}</span>
属性
如果绑定的值是null或者undefined，那么被绑定的元素的值将被移除
使用方式：v-bind
说明：v-bind 指令指示 Vue 将元素的 id attribute 与组件的 dynamicId 属性保持一致。
一般示例
详写
<div v-bind:id="dynamicId" :div="dynamicDiv"></div>
简写
<div :id="dynamicId"></div>
同名简写（3.4+），当attribute 的名称与绑定的 JavaScript 值的名称相同
<!-- 与 :id="id" 相同 -->
<div :id></div>

<!-- 这也同样有效 -->
<div v-bind:id></div>
布尔型
说明：依据 true / false 值来决定 attribute 是否应该存在于该元素上。
示例
disabled
值为true或空字符串：元素会包含disabled属性
值为其他时：元素会忽略disabled属性

动态绑定多个值
如果你有像这样的一个包含多个 attribute 的 JavaScript 对象：
data() {
  return {
    objectOfAttrs: {
      id: 'container',
      class: 'wrapper'
    }
  }
}
通过不带参数的 v-bind，你可以将它们绑定到单个元素上：
<div v-bind="objectOfAttrs"></div>
class
绑定对象
下列的语法表示 active 是否存在取决于数据属性 isActive 的真假值。
<div :class="{ active: isActive }"></div>
绑定数组
vue代码
data() {
  return {
    activeClass: 'active',
    errorClass: 'text-danger'
  }
}
html
<div :class="[activeClass, errorClass]"></div>
渲染结果
<div class="active text-danger"></div>
style
1. vue会自动为css属性添加相应的浏览器特殊前缀
vue
data() {
  return {
    styleObject: {
      color: 'red',
      fontSize: '13px'
    }
  }
}
html
<div :style="styleObject"></div>
HTML
请勿动态渲染任意HTML，容易造成XSS漏洞，不要向用户提供HTML内容
使用方式：v-html
示例
<p>Using text interpolation: {{ rawHtml }}</p>
<p>Using v-html directive: <span v-html="rawHtml"></span></p>
访问限制
仅能够访问到有限的全局对象列表，比如内置的Math 和 Date。
