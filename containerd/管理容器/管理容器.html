状态
启动指定的容器：
ctr container start <容器ID>
停止指定的容器：
ctr container stop <容器ID>
强制终止指定的容器：
ctr container kill <容器ID>
暂停指定的容器：
ctr container pause <容器ID>
恢复暂停的容器：
ctr container resume <容器ID>
列出所有容器的信息：
ctr containers list
显示指定容器的详细信息：
ctr container info <容器ID>
删除指定的容器：
ctr container delete <容器ID>
查看指定容器的日志：
ctr container logs <容器ID>
在容器中执行指定的命令：
ctr container exec <容器ID> <命令>
更新容器的配置：
ctr container update <容器ID> <配置文件>
